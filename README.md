# Phishing Email Detection & Awareness

## Overview
This project is part of my Cyber Security Internship (Task 2).
The objective of this task is to analyze phishing emails, identify
common phishing indicators, and spread awareness about email-based
cyber attacks.

## Objectives
- Understand phishing email techniques
- Analyze suspicious email samples
- Classify emails as Safe, Suspicious, or Phishing
- Provide awareness and prevention tips

## Tools & Techniques Used
- Sample phishing emails
- Manual email content analysis
- Basic email header analysis
- Web browser link inspection
- MS Word (for report creation)
